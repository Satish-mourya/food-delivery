## Project Overview

Tomato is a modern full-stack food delivery web application developed using the MERN stack (MongoDB, Express.js, React.js, and Node.js). The platform is designed to provide users with a seamless online food ordering experience while offering restaurant administrators a powerful dashboard for managing menu items and customer orders.

The application allows users to browse a wide range of food categories, explore detailed menu items, and place orders through an intuitive and responsive interface. Customers can create accounts, securely log in, manage their carts, and complete purchases using integrated online payment functionality. The platform emphasizes simplicity, performance, and accessibility, ensuring a smooth experience across desktops, tablets, and mobile devices.

One of the key objectives of this project was to simulate a real-world food delivery system and implement industry-standard development practices. The application follows a client-server architecture, where the frontend communicates with RESTful backend APIs to fetch data, manage user sessions, and process orders. Images are stored and managed using cloud storage services, while MongoDB provides scalable and efficient data management for users, food items, carts, and orders.

The project includes a dedicated admin panel that enables restaurant administrators to add, update, and remove food items, upload food images, and monitor incoming orders. This separation between customer and administrative functionalities creates a complete end-to-end food delivery ecosystem.
